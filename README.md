
# 信用卡管理工具，计算所有卡片总额度、总可用额度、总已用额度。

# 软件介绍：剪卡

## 产品概述

**剪卡**是一款专为信用卡用户打造的智能管理小程序，口号为"上岸第一剑·先剪手中卡"。产品致力于帮助用户科学管理多张信用卡，通过AI智能识别和自动化记录，让信用卡管理更加轻松高效。

## 核心功能

### 1. 🎴 信用卡管理
- 支持多张银行卡和信用卡的集中管理
- 实时查看卡片额度、可用余额、已用额度
- 卡片详情查看与编辑
- 支持按额度、负债等维度排序展示
- 可视化展示卡片使用率

### 2. 🤖 AI智能记账
- 剪切板智能识别：自动检测并识别剪切板中的消费记录
- AI智能解析：通过AI自动提取消费金额、商户、时间等关键信息
- 一键快速记账：无需手动输入，大幅提升记账效率
- 支持自定义识别关键词

### 3. 📊 账单管理
- 信用卡账单自动记录（支持AI识别）
- 账单还款日期提醒
- 本月账单汇总统计
- 账单详情查看与编辑

### 4. 📈 数据统计与报表
- 综合资产统计：总额度、总负债、总余额一目了然
- 交易流水时间线：清晰展示历史记录
- 数据分析报表：多维度分析消费情况
- 打卡功能：记录上岸天数

### 5. 🗓️ 日历视图
- 交易日历展示
- 按日期查看消费记录
- 直观呈现消费分布

### 6. 🤝 用户系统
- 账号注册与登录
- 个人信息管理
- 意见反馈通道
- 分享邀请功能

## 产品特色

### ⚡ 智能化体验
- **AI智能识别**：利用AI技术自动识别银行短信和账单信息
- **自动记账**：开启剪切板监听后，复制消费信息即可自动记账
- **智能推荐**：AI助手提供使用建议

### 🔄 双模式切换
- **上岸模式**：优先展示负债情况，帮助用户控制消费
- **刷卡模式**：侧重展示消费记录，方便查看交易明细

### 💎 便捷操作
- 简洁直观的界面设计
- TDesign组件库提供流畅交互
- 快速添加记录，一键完成操作

## 适用场景

- ✅ 多张信用卡需要集中管理的用户
- ✅ 希望自动记录消费和账单的用户
- ✅ 需要控制信用卡负债的用户
- ✅ 上岸途中需要财务管理的用户
- ✅ 希望通过AI简化记账流程的用户
